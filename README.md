# ichunhui.github.io
[chunhui's blog](http://ichunhui.github.io/)

ichunhui.github.io 博客的hexo源码，包括主题、配置文件等

当重装电脑之后，或者想在其他电脑上修改博客，可以使用下列步骤：

安装git，nodejs；

使用git clone git@github.com:ichunhui/mybloggit拷贝仓库；

在本地新拷贝的myblog文件夹下通过Git bash依次执行下列指令：
npm install

hexo clean

hexo generate

hexo deploy

hexo server
